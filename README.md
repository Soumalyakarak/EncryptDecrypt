# EncryptDecrypt

A high-performance concurrent file encryption and decryption system built in modern C++ using AES-256-GCM.

The project explores multiprocessing and multithreading approaches for parallel cryptographic execution.

> The latest stable implementation is available in the `v1/EncryptDecrypt` branch.

## Features

- AES-256-GCM authenticated encryption
- Parallel file processing
- Multithreading and multiprocessing
- Benchmarking and performance analysis
